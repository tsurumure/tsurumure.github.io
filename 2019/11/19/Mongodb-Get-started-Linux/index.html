安装 Mongodb

MongoDB shell version v4.0.10
1.下载: https://fastdl.mongodb.org/linux/mongodb-linux-x86_64-4.0.10.tgz
2.通过 FTP 软件上传到自定义目录 (例如 /home/www)
3.解压并修改目录名称

4.设置环境变量 (软链接)

5.开启服务

更多步骤参考：https://www.runoob.com/mongodb/mongodb-linux-install.html
