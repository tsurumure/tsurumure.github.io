Mongodb 安装与配置 (Window)

发表于 2019-11-19 分类于 Mongodb 阅读次数： Valine：

安装 Mongodb

https://www.mongodb.com/download-center/community
下载系统对应版本 (mongodb-win32-x86_64-2008plus-ssl-4.0.10-signed.msi) 并安装
注意：安装目录不能是 Program Files\ 之类带有空格名称的文件夹

  • 安装目录：D:\MongoDB\Server\4.0\
  • 不要勾选 compass，另外单独安装，否则会因为国外服务器下载时间太长导致卡住

安装 Mongodb-Compass (可视化界面)

由于外网访问速度慢，有时候下载不通，可以在百度寻找其他低版本资源下载
https://www.mongodb.com/download-center/compass
